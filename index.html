br://web

O script.js cuida apenas da lógica de navegação. Você é livre para criar qualquer visual que quiser, siga o guia abaixo! :)

1. Importe o script

Basta adicionar a tag abaixo no final do seu <body> 

<script src="https://hikilune.github.io/webring/script.js"></script>

2. Crie a estrutura HTML

Seu frontend só precisa:

  • Um elemento com classe brwebring
  • Um link com classe prev
  • Um link com classe rand
  • Um link com classe next

Exemplo mínimo:

<div class="brwebring">
  <a class="prev" href="#">Prev</a>
  <a class="rand" href="#">Random</a>
  <a class="next" href="#">Next</a>
</div>

3. Personalize do seu jeito

O script não interfere no visual. Use os seletores abaixo no seu CSS para estilizar do seu jeito: 

.brwebring { text-align: center; }

.brwebring .prev   { color: #2e7a4f; } /* verde  */
.brwebring .rand   { color: #b8921a; } /* amarelo */
.brwebring .next   { color: #1a4b7a; } /* azul   */
